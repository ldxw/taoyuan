### 实时钓鱼小游戏机制

抛竿后进入实时小游戏, 与星露谷原版机制类似:

**视觉布局**: 竖向钓鱼条 (250px), 左侧进度条, 右侧倒计时+分数

| 要素 | 说明 |
|------|------|
| 钓鱼区 | 宽40px × 高250px, 鱼(金色块)上下随机移动 |
| 钩子(绿条) | 自动下落, 长按按钮/空格/↑键上升 |
| 进度 | 钩子覆盖鱼时 +scoreGain/帧, 否则 -scoreLoss/帧 |
| 成功 | 进度达 100% |
| 超时 | 倒计时归零, 按进度评级 |

**小游戏评级**:

| 评级 | 条件 | 效果 |
|------|------|------|
| 完美 (perfect) | 进度达 100% 且从未下降 | 品质+2档, 经验×2 |
| 优秀 (excellent) | 进度达 100% | 品质+1档 |
| 良好 (good) | 超时但进度 ≥ 60% | 正常捕获 |
| 失败 (poor) | 超时且进度 < 60% | 鱼跑掉 |

**鱼的运动算法**:
```
fishVelocity += (targetDirection - fishVelocity) × 0.05  // 缓动
fishPosition += fishVelocity
// 以 fishChangeDir 概率随机改变方向
if (random < fishChangeDir) targetDirection = random × fishSpeed × 2
// 边界反弹
```

**每鱼独立行为参数**: 全部 61 种鱼均有独立的 `miniGameSpeed` 和 `miniGameDirChange`, 不再使用统一难度默认值。

| 难度 | 速度范围 | 方向变化范围 | 代表鱼 (速度/方向变化) |
|------|----------|-------------|----------------------|
| Easy | 0.5 ~ 1.5 | 0.01 ~ 0.04 | 田螺(0.5/0.01) 最慢, 溪哥鱼(1.3/0.04) 最灵活 |
| Normal | 1.0 ~ 2.8 | 0.02 ~ 0.08 | 乌龟(1.0/0.02) 极慢, 刀鱼(2.8/0.03) 极快, 蛙鱼(2.0/0.08) 最飘忽 |
| Hard | 2.0 ~ 3.5 | 0.03 ~ 0.09 | 鲟鱼(2.5/0.03) 稳重, 河鳗(3.5/0.07) 滑溜 |
| Legendary | 2.0 ~ 5.5 | 0.05 ~ 0.15 | 深渊巨蟒(4.5/0.15) 最难, 娃娃鱼(2.0/0.10) 慢但诡异 |
